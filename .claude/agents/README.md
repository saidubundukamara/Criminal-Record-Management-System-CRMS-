## How to Use Agents

### In Claude Code CLI

When working with Claude Code, you can reference these agents in your prompts:

```bash
# Example: Architecture guidance
claude "Act as the CRMS Architect Agent. Help me implement a new Evidence Transfer feature following the Service-Repository pattern."

# Example: Security review
claude "Act as the CRMS Security Auditor Agent. Review the authentication system for potential vulnerabilities."

# Example: Testing strategy
claude "Act as the CRMS Tester Agent. Create a comprehensive test plan for the offline synchronization feature."
```
